⚠️ RESTAURAR BACKUP GERAL

Restaura o programa inteiro (servidor, páginas, código e dados) a partir de um arquivo gerado pelo botão "Backup Geral" (.zip).

⚠ Depois de restaurar, é preciso reiniciar o servidor manualmente (ex: Ctrl+C e "npm start", ou "pm2 restart") para que mudanças no código do servidor tenham efeito. As páginas (HTML/CSS/JS) já aparecem atualizadas no próximo carregamento, mas o processo do servidor continua rodando o código antigo até reiniciar.
